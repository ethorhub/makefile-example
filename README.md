## Configure
```bash
./configure
```

## Build make version
```bash
make build-v1
```

## Build
```bash
make build
```

## Usage
```bash
make
```
or run with static libraries
```bash
make run_static
```

## Clean
```bash
make clean
```
force clean
```bash
make clean FORCE=true
```

## Default make
```bash
make default-make
```
